Architecture Overview

Simple control-plane to execution-plane flow.

The Web UI and API act as the control plane for configuration and permissions, while the workflow engine and connected AI, voice, RAG, and provider systems handle execution.

Web app

Operators manage workspaces, integrations, saved data, workflows, and run visibility.

API layer

The API validates auth, resolves workspace scope, and proxies orchestration requests safely.

Workflow engine

The engine executes workflow steps, agentic actions, analytics capture, and provider-specific runtime logic.

Integrations

External systems such as Google OAuth, OpenAI, voice endpoints, RAG sources, webhooks, and internal APIs connect per workspace.

High-level flow: Web UI -> API -> Workflow Engine -> AI / Voice / RAG / Integrations
